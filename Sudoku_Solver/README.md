# Sudoku Solver Game 🧩

A GUI-based Sudoku game built using Python and Tkinter with backtracking algorithm.

## Features
- 🎯 Sudoku puzzle generator
- 🧠 Backtracking solver
- ✅ Input validation (row, column, 3x3 rules)
- 🎮 Interactive UI using Tkinter
- 💡 Hint system
- ❌ Error highlighting
- 🔢 Attempt-based checking system

## Tech Stack
- Python
- Tkinter
- Algorithms (Backtracking)

## How to Run
```bash
python main.py
